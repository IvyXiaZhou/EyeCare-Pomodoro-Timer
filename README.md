EyeCare Pomodoro Timer
A desktop application that combines the Pomodoro Technique with eye care reminders, helping you maintain productivity while protecting your eye health.
Core Features
•	Smart Timing: Optimised 40-minute work + 10-minute break cycle
•	Visual Eye Protection: Soft green color scheme to reduce eye strain
•	Progress Visualisation: Real-time display of remaining time and progress bar
•	One-Click Operation: Simple and intuitive start, pause, reset functions
•	Data Statistics: Automatic daily work cycle tracking
System Requirements
•	Python 3.7 or higher
•	Supports Windows, macOS, Linux
Usage Guide
1.	Launch App: After running the program, the main interface displays the timer
2.	Start Working: Click the "Start" button to begin the 40-minute work timer
3.	Break Reminder: Automatically switches to 10-minute break mode after work period
4.	View Statistics: The program automatically records daily completed work cycles

护眼番茄钟
一款结合番茄工作法与护眼提醒的桌面应用程序，帮助你在保持工作效率的同时保护视力健康。
核心功能 
•	智能计时：40分钟工作 + 10分钟休息的优化时长设定
•	视觉护眼：采用柔和的绿色系配色，减少视觉疲劳
•	进度可视化：实时显示剩余时间和进度条
•	一键操作：简单直观的开始、暂停、重置功能
•	数据统计：自动记录每日完成的工作周期
系统要求 
•	Python 3.7 或更高版本 
•	支持 Windows、macOS、Linux
使用方法 
1.	启动应用：运行程序后，主界面将显示计时器
2.	开始工作：点击"开始"按钮，进入40分钟工作计时
3.	休息提醒：工作结束后自动进入10分钟休息模式
4.	查看统计：程序自动记录每日完成的工作周期数
______________________________________
健康提示: 记得定时休息，保护眼睛健康！
